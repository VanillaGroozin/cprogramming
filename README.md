2. Переменная
    * Переменная - имя кусочка памяти, определенного типа.
    * Синтаксис объявления переменной: `type name;`
      - пример: `int a;`, `int a, b;`
    * Инициализация переменной - задание первого значения.
      - пример: `int a = 5;`, `int a=5, b=7;`
